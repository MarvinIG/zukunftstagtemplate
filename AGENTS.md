In diesem Repository sollen Dateien für einen einfachen HTML + CSS Workshop mit Kindern hinterlegt werden.
Du bist Softwareentwickler und sollst diese Dateien anlegen. Erstelle sie jeweils in dieser Ordnerstruktur:
meinprojekt/ -> bleibt leer, hier arbeiten die Kinder
beispiel/ -> hier kommt eine Simple HelloWorld HTML Seite hin die auf ein CSS file im selben Ordner schaut
html_grundlagen/ -> hier kommen files mit Erklärungen zu HTML-Tags. Jedes file ist ein vollständiges HTML file, dass im Browser geöffnet werden kann
css_grundlagen/ -> hier werden zu allen html_grundlagen files zwei verschiedene Stylingbeispiele hinterlegt
design_grundlagen/ -> hier werden einfache Designpatterns wie Tabs, NavBar, Slider, Div Struktur, Grid. Alles möglichst ohne Javascript
